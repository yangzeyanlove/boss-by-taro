# boss-by-taro

使用 taro，模仿 boss 微信小程序

# 使用技术栈

- taro3.x
- react
- taro-ui
- typescript
- mobx 做状态管理

# 项目目录结构

```js
src/                  // - 这是存放所有应用程序逻辑的地方，通常包括以下子目录：
  assets/             // - 存放所有的静态资源，比如图片、字体文件等。
    fonts/            // - 字体资源。
    images/           // - 图片资源。
  common/             // - 常用工具类，例如请求封装，缓存封装等等
  components/         // - 公共的组件
  config/             // - 配置文件
  mobx-store/         // - 状态管理
  pages/              // - 所有页面的入口文件
  app.config.ts;      // - 小程序相关的配置文件
  app.ts;             // - 应用程序入口点文件。
types/                // - typescript公共类型
```
